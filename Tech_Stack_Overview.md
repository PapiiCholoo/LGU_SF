# Project Tech Stack Overview

## Frontend Application
**Core Technologies**
- **Language:** TypeScript
- **Framework:** React 18
- **Build Tool:** Vite
- **Package Manager:** npm

**UI & Styling**
- **Styling Engine:** Tailwind CSS (v4)
- **Component Primitives:** Radix UI
- **Icons:** Lucide React
- **Utility Libraries:** `clsx`, `tailwind-merge`, `class-variance-authority` (Indicating usage of shadcn/ui patterns)

**Feature Libraries**
- **Form Management:** React Hook Form
- **Data Visualization:** Recharts
- **Carousel/Slider:** Embla Carousel
- **Notifications:** Sonner
- **Date/Time:** React Day Picker

## Backend API
**Core Technologies**
- **Runtime:** Node.js
- **Framework:** Express.js
- **Language:** TypeScript

**Database & Storage**
- **Database:** MySQL
- **ORM:** Prisma (v5.7.0)

**Security & Infrastructure**
- **Authentication:** JSON Web Token (JWT)
- **Encryption:** Bcrypt
- **Validation:** Zod
- **Security Headers:** Helmet
- **CORS Support:** cors
- **Logging:** Morgan
